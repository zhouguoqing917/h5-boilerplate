对话框 dialog

  • 基础样式
  • UI组件
  • JS插件

创建一个对话框，不加遮罩

            $.rDialog({
                    content : '对话框内容',
                    title : 'ok',
                    ok : function() {
                        alert('我是确定按钮，回调函数返回false时不会关闭对话框。');
                        return false;
                    },
                    cancel : function() {
                        alert('我是取消按钮');
                    },
                    lock : true
                });

运行

2秒自动关闭

            $.rDialog({
                    content : '窗口将在2秒后自动关闭',
                    title: "alert",
					width: 600,
                    time : 2000
                });

运行

加载中，1秒后消失

             $.rDialog();
